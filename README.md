<h2> 🔍 Comentários </h2>
-> 💯 Sobre o projeto:

Elaboração de um projeto website de vendas para ingressos, com uma quantidade definida por um comprador em função da quantidade disponível para o publico, com vários tipos de ingressos distintos para ampla escolha.

-> 🚨 Melhorias:

Melhorias na quantidade inserida, caso o comprador dígite o valor 0 ou nenhum valor para a quantidade será enviada uma notificação pela tela, para cada respectiva escolha de ingressos e sua respectiva quantidade será subtraida do valor de ingressos disponíveis e será atualizada imediatamente. Caso a quantidade inserida seja maior que a quantidade disponível será enviada uma notificação para o comprador com uma mensagem "Não há ingressos suficientes".
